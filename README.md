# CryptoBot

CryptoBot allows you to monitor cryptocurrency prices and also set up a donation portfolio to accept cryptocurrencies from other members.
Invite bot to your Discord Server [here](https://discord.com/api/oauth2/authorize?client_id=953953187394617354&permissions=274877908992&scope=bot%20applications.commands).

## CryptoCurrencies

All 2500+ cryptocurrencies and 160+ fiat currencies provided by [RabbitForexAPI](https://docs.forex.rabbitmonitor.com/) are supported via autocomplete.

## Commands

- `/help` - Show all commands
- `/price [symbol]` - Show price of a specific crypto
- `/price [symbol] [amount] [currency]` - Calculate the fiat value of a specific amount of crypto assets
- `/convert [amount] [from] [to]` - Convert between any crypto and fiat currencies
- `/address set [symbol] [address]` - Set crypto address in your donation list
- `/address remove [symbol]` - Remove crypto address from your donation list
- `/address list` - Show your own donation addresses
- `/donate [user]` - Show donation list from a specific user
- `/whitelist add [channel]` - Add channel to the whitelist
- `/whitelist remove [channel]` - Remove channel from the whitelist
- `/vote` - Support the bot with daily voting
- `/privacy` - Show Privacy Policy
